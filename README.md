## 1. Prerequisites

- Ubuntu 20.04 and ROS Neotic
- PCL >= 1.10
- Eigen >= 3.3.7 
- GTSAM >= 4.0.3

## 2. Build

```bash
cd YOUR_WORKSPACE/src
git clone 
cd ..
source livox_driver/devel/setup.bash
catkin_make
```
